<h1>Spring boot Workout tracker service</h1> 
<p>Um gerenciador de treinos para academia</p>

<ol>
    <li> usuario loga na aplicacao
    <li> Usuario tem acesso aos seus planos de treino
    <li> usuario pode criar novo plano
    <ol>
        <li> da um nome para o treino (Treino de perna, cardio, matinal e etc)
        <li> define a recorrencia desse exercício (uma vez, diariamente, semanalmente e mensalmente)
        <li> Define a data do treino
        <li> Seleciona os exercícios relacionados ao treino
        <ol>
            <li> pegar exercícios gerais já prontos ou
            <li> Criar seus proprios exercícios
            <li> Colocar as reps e series
        </ol>
    </ol>
    <li> ALterar o estado do treino (Completo ou pendente)
    <li> deletar e atualizar exercícios e treinos

</ol>
